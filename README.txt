== Description ==

Ce plugin permet d'afficher les review collectés auprès des clients et de les afficher dans un carousel.

Ce carousel est appelé à l'aide d'un shortcode. 

Il faudra  placer le shortcode suivant : [hb_carousel_review]  et le carousel sera affiché!

A la base le plugin est censé faire Appel à l'API de TRUSTPILOT.fr qui est le site collectant les avis clients.
Cependant, comme l'accès à l'API est payant, ce plugin offre simplement une page avec un formulaire pour ajouter les avis qui seront copié et collé depuis le site (ou non...)

la page d'administration offre une liste des avis avec la possibilité de les éditer , effacer .
Puis un accès à une page pour ajouter un avis (CRUD COMPLET).

=== installation ===

Deux possibilité :
1 - placer le dossier dans le répertoire 'wp-content/plugin/s' puis activer depuis l'administration

2 - archiver le dossier dans un fichier zip, puis ajouter l'extension depuis le dashboard d'administration de worpdress.
